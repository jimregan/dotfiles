Getting Started

  • Download some existing data from OSM using Download from OSM on the File menu or the download button (green downwards pointing arrow on the toolbar)
  • Your username and password are sent to the server unencrypted. If you do not like this, you may consider to use OAuth.

Getting Help

  • Online help: Help on the Help menu or press F1 as required to get context sensitive help. Open Help now.
  • Join the  forum or the  Newbies mailing list

Remember the golden rules of OpenStreetMap:

  • Don't copy from other maps
  • Have fun!
